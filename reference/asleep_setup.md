# Check the `asleep` Python Module

Check the `asleep` Python Module

## Usage

``` r
have_asleep()

asleep_check()

asleep_version()
```

## Value

A logical value indicating whether the `asleep` Python module is
available.

## Examples

``` r
if (have_asleep()) {
   asleep_version()
}
#> [1] "0.4.16"
```
